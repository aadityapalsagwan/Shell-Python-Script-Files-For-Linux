# Shell Python Script Files For Linux

This repository contains useful Shell and Python scripts for Linux system administration and automation tasks.

## Available Scripts

### install-apache-webserver.sh
This shell script installs and configures the Apache web server on Linux systems.

## Requirements

- Linux OS
- Bash shell
- Sudo privileges

## Usage

Make the script executable:

```bash
chmod +x install-apache-webserver.sh
